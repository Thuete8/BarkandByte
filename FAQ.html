Häufig gestellte Fragen


Welche Kriterien sind für die Punktemarkierung im Spiel vorgesehen und wie sollen diese Informationen genutzt werden, um den Punktestand zu übermitteln?

Man kann nur Punkte erhalten, indem man eine Figur in ein Haus zieht, die Punkte erhält man durch ein Interface


Soll das Spiel durch eine maximale Anzahl von Zügen und/oder eine maximale Spieldauer begrenzt werden, oder sollen sowohl die Gesamtzahl der Züge als auch die maximale Spieldauer unbegrenzt sein?

Die Gesamtzahl der Züge als auch die maximale Spieldauer ist unbegrenzt


Wer hat die Autorität und Befugnis, den Engine-Teilnehmer zu starten, und welche Schritte sind in der Implementierung vorgesehen, um den Engine-Teilnehmer erfolgreich einem Spiel hinzuzufügen?

Ja, das können sie wenn sie wollen


Im Falle eines Spielabbruchs möchten wir Klarheit darüber erhalten, wer das Spiel gewinnt. Auf welcher Grundlage wird die Wertung vorgenommen?

Ja, das können sie wenn sie wollen


Kann der Organisator die Größe des Spielfelds anhand der Anzahl der Felder zwischen den Startfeldern variieren?

Ja, das können sie wenn sie wollen


Muss ein Engine-Teilnehmer übernehmen, wenn ein realer Spieler sich abmeldet oder einen Fehler macht? Hat der betroffene Spieler die Möglichkeit, wieder in das Spiel einzusteigen? Und wie wird das Spiel in einer solchen Situation bewertet?

Ja, ein Engine-Teilnehmer übernimmt. Der Betroffene kann nicht wieder ins Spiel einsteigen. Der Ausrichter legt die Reihenfolge fest.

Nein, es gibt keine Begrenzung der Figuren pro Spieler
